# 🛒 Blinkit Sales Analysis using Python

## 📌 Project Overview
This project analyzes Blinkit sales data using Python to understand customer purchasing behavior, sales trends, and business performance. The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization.

---

## 🎯 Objectives
- Clean and preprocess the dataset
- Analyze sales performance
- Identify customer purchasing patterns
- Visualize important business insights
- Generate meaningful recommendations

---

## 📂 Dataset
The dataset contains information such as:
- Item Identifier
- Item Type
- Outlet Identifier
- Outlet Size
- Outlet Location Type
- Outlet Type
- Item Outlet Sales
- Outlet Establishment Year
- Item Visibility
- Item Rate
- Item Fat Content

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Handle Missing Values
5. Exploratory Data Analysis (EDA)
6. Data Visualization
7. Business Insights

---

## 📈 Key Insights
- Identified top-selling product categories.
- Compared sales across different outlet types.
- Analyzed the impact of outlet size on sales.
- Studied customer purchasing trends.
- Visualized sales distribution and relationships.
